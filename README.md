# NTab — Chrome 新标签页扩展

替换浏览器默认新标签页，展示常用网站、搜索引擎和访问排行。

## 功能

### 搜索栏
- **多引擎切换** — Google / Bing / 百度，下拉菜单选择，带品牌色图标
- **搜索建议** — Google / Bing 输入时实时获取搜索引擎联想词
- **书签 & 历史搜索** — 输入关键词即时匹配本地书签和历史记录
- **智能 URL 识别** — 输入网址直接跳转，非网址则用当前引擎搜索
- **两步确认** — 从下拉选项填入的内容需二次回车才执行搜索/跳转
- **即时过滤** — 搜索时同步过滤页面上的网站卡片

### 网站卡片
- **8×5 网格** — 最多 40 个网站卡片，响应式适配
- **Chrome 风格** — 圆角方形图标 + favicon，回退显示彩色首字母
- **固定/锁定** — 手动添加的卡片可固定位置，锁定后不可拖拽
- **拖拽排序** — 未锁定卡片支持拖拽换位
- **分组标签** — 支持拖拽分组和卡片菜单手动修改所属分组
- **三点菜单** — 重命名、编辑网址、固定/取消固定、移除

### 使用统计
- 基于浏览历史统计今天 / 7 天 / 30 天的网站访问情况
- 展示访问次数、活跃站点、最常访问网站和较上一周期趋势
- 展示 TOP 网站排行和工作 / 学习 / 娱乐 / 工具分类占比
- 点击排行网站可直接跳转，统计仅在本地生成

### 深色模式
- CSS 变量驱动，跟随系统 `prefers-color-scheme`
- 右上角手动切换按钮，无闪烁切换

### 数据持久化
- `chrome.storage.local` 保存固定卡片、隐藏列表、搜索引擎偏好、主题偏好
- 动态网站来源 — `chrome.topSites` + 历史记录高频域名自动填充

## 安装

1. 下载 `myNewTab.zip` 并解压
2. 打开 Chrome，进入 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择解压后的文件夹

## 权限

默认权限：`topSites` / `favicon` / `storage`

可选权限：`history` / `bookmarks`，用于使用统计、历史记录补充和本地书签/历史搜索。

## 技术栈

Manifest V3，零外部依赖，纯 HTML/CSS/JS 实现。
