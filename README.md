# MyTab — Chrome 新标签页扩展

替换浏览器默认新标签页，集成搜索、常用网站、访问统计与快捷便签。

## 功能

### 时钟
- 显示年月日 + 星期 + 时分秒，秒级自动刷新
- 页面隐藏时自动暂停，恢复可见时补跳至当前时间

### 搜索栏
- **多引擎切换** — Google / Bing / 百度，下拉菜单选择，带品牌色图标
- **搜索建议** — 输入时实时获取搜索引擎联想词（Google / Bing）
- **书签 & 历史搜索** — 输入关键词即时匹配本地书签和历史记录
- **智能 URL 识别** — 输入网址直接跳转，非网址则用当前引擎搜索
- **两步确认** — 从下拉建议填入的内容需二次回车才执行搜索/跳转
- **即时过滤** — 搜索时同步过滤页面上的网站卡片

### 网站卡片
- **8×5 网格** — 最多 40 个网站卡片，响应式布局
- **数据来源** — `chrome.topSites` + 历史记录高频域名，Frecency 时效加权排序（5 天半衰期）
- **Chrome 风格图标** — 圆角方形 + favicon，回退显示彩色首字母
- **固定/锁定** — 手动添加的卡片可固定位置，锁定后不可拖拽
- **拖拽排序** — 未锁定卡片支持拖拽换位，空卡位也可作为拖拽目标
- **分组标签** — 全部 / 工作 / 学习 / 工具 / 其他，支持拖拽分组和右键菜单手动分组
- **右键菜单** — 重命名、编辑网址、分组管理、固定/取消固定、移除
- **空卡位占位** — 空位显示虚线边框占位提示
- **手动刷新** — 一键重新获取最新网站数据
- **中键/Ctrl+点击** — 在新标签页后台打开链接

### 使用统计
- 基于浏览历史统计今天 / 7 天 / 30 天的网站访问情况
- 展示总访问次数、活跃站点数、最常访问网站和较上一周期趋势
- TOP 网站排行，点击可直接跳转
- 工作 / 学习 / 娱乐 / 工具分类占比统计
- 统计仅在本地生成，数据不上传

### 快捷便签
- 右下角 FAB 按钮打开/关闭便签面板
- 最多 5000 字，自动保存至本地存储
- 未保存内容显示红点提示

### 深色模式
- CSS 变量驱动，跟随系统 `prefers-color-scheme`
- 右上角手动切换按钮，偏好持久化
- 无闪烁切换

### 数据持久化
- `chrome.storage.local` 保存固定卡片、隐藏列表、分组配置、搜索引擎偏好、主题偏好、便签内容
- 动态网站来源 — `chrome.topSites` + 历史记录高频域名自动填充

## 安装

1. 下载 `3ms-终版.zip` 并解压
2. 打开 Chrome，进入 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择解压后的文件夹

## 权限

| 权限 | 类型 | 用途 |
|------|------|------|
| `topSites` | 默认 | 获取常访问网站 |
| `favicon` | 默认 | 获取网站图标 |
| `storage` | 默认 | 本地数据持久化 |
| `history` | 可选 | 使用统计、历史记录补充搜索 |
| `bookmarks` | 可选 | 本地书签搜索 |

## 技术栈

Manifest V3，零外部依赖，纯 HTML/CSS/JS 实现。
